# 🎂 Birthday Letters

HTML-файл для печати поздравления **«С ДНЁМ РОЖДЕНИЯ КАТЯ И ВАНЯ!»** большими буквами на бумаге.

## Как устроено

- Каждая буква печатается в формате **A6**
- На одном листе **A4** помещается **4 буквы** (сетка 2×2)
- Итого **6 листов** с буквами + 1 лист с надписью целиком + 1 лист-шаблон для разрезания
- Шрифт: **JetBrains Mono ExtraBold**
- Ориентация: **портретная**

## Цвета

Буквы раскрашены в пастельные тона, соседние буквы — всегда разного цвета:

| Группа | Цвета |
|--------|-------|
| «С ДНЁМ РОЖДЕНИЯ!» | голубой → жёлтый → зелёный → красный (циклично) |
| «КАТЯ» | красный / зелёный |
| «И» | жёлтый |
| «ВАНЯ» | голубой / жёлтый |

## Как печатать

1. Откройте `birthday-letters.html` в браузере (нужен интернет для загрузки шрифта)
2. `Ctrl + P` → ориентация **Книжная (Portrait)** → поля **Нет**
3. Напечатайте **страницы 2–7** — буквы
4. Положите листы обратно в принтер обратной стороной вверх
5. Напечатайте только **страницу 8** — шаблон разрезания окажется на обороте
6. Разрежьте каждый лист по пунктирному кресту на **4 части A6**
7. Разложите буквы в нужном порядке и повесьте на стену 🎉
